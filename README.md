# 海看AI培训战略汇报平台

🏢 **海看AI培训：构建"技术+内容+场景"新生态的工作汇报**

一个现代化的交互式可视化汇报平台，完整呈现110行战略规划内容，为领导层提供清晰、直观的决策支持。

## ✨ 项目特色

- 📖 **完整保留原文** - 110行稿件内容一字不删，全部可通过交互方式访问
- 🎯 **智能导航系统** - 分层级内容导航，支持快速定位和跳转
- 🔍 **全文搜索功能** - 支持关键词搜索、内容筛选和高亮显示
- 📊 **数据可视化** - 时间轴、矩阵图、进度条等多种图表展示
- 📌 **书签标注系统** - 支持添加书签、标注，记录重要内容
- 📱 **响应式设计** - 完美适配桌面、平板、手机等设备
- 🎨 **专业商务风格** - 简洁大气的设计，适合企业汇报场景
- 📤 **多格式导出** - 支持PDF、Word、JSON等多种格式导出

## 🚀 快速开始

### 环境要求

- Node.js 16.0.0 或更高版本
- npm 8.0.0 或更高版本

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

启动后访问 [http://localhost:5173](http://localhost:5173)

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 🎯 使用指南

### 基本操作
1. **浏览内容** - 从首页开始，点击导航卡片进入各章节
2. **搜索内容** - 使用搜索框快速查找关键词
3. **添加书签** - 点击行号旁的操作按钮添加书签
4. **添加标注** - 为重要内容添加个人标注
5. **导出报告** - 生成PDF或Word格式的报告

### 高级功能
1. **完整内容浏览** - 查看所有110行原文的完整列表
2. **筛选查看** - 按内容类型筛选显示特定内容
3. **行级定位** - 通过URL直接跳转到指定行
4. **批量操作** - 展开/收起全部内容
5. **数据导出** - 导出结构化数据用于分析

### 快捷键
- `Ctrl/Cmd + F` - 打开搜索面板
- `Ctrl/Cmd + K` - 快速搜索
- `Esc` - 关闭弹窗
- `Home` - 返回顶部

## 🔧 自定义配置

### 主题设置
在 `src/styles/main.css` 中修改CSS变量：
```css
:root {
  --primary-color: #1e3a8a;  /* 主色调 */
  --success-color: #059669;  /* 成功色 */
  --warning-color: #d97706;  /* 警告色 */
  --danger-color: #dc2626;   /* 危险色 */
}
```

### 内容更新
修改 `src/data/` 目录下的JSON文件：
- `content.json` - 更新原文内容
- `structure.json` - 调整内容结构
- `metadata.json` - 修改元数据

### 功能扩展
- 在 `src/components/` 中添加新组件
- 在 `src/views/` 中添加新页面
- 在 `src/utils/` 中添加新工具函数
